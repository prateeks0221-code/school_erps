# School ERP System

A modern, scalable Enterprise Resource Planning (ERP) solution designed for schools and educational institutions.

## ✨ Features

- **Student Management** - Complete student information, enrollment, and academic records
- **Grades & Attendance** - Track attendance and manage grades efficiently
- **Staff Portal** - Teacher dashboards and resource management
- **Analytics & Reports** - Comprehensive analytics and custom reports
- **Responsive Design** - Mobile-friendly interface for all devices

## 🚀 Quick Start

### Prerequisites

- Git installed on your system
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/school_erps.git
   cd school_erps
   ```

2. View the demo:
   ```bash
   # On Windows
   start src/demo-reel.html
   
   # On macOS
   open src/demo-reel.html
   
   # On Linux
   xdg-open src/demo-reel.html
   ```

### Running the Demo

Simply open `src/demo-reel.html` in your web browser to view the School ERP system demo.

```bash
cd src
# Then open demo-reel.html with your favorite browser
```
